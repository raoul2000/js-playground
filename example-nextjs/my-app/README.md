## Getting Started

### Configuration

Require environment variables (in dev, create file `.env.local` at the project's root folder) : 

```
NEXTAUTH_SECRET=************
GITHUB_SECRET=************
GITHUB_ID=***************
```

run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
